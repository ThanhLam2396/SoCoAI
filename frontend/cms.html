Quick Guide to Customizing the AI Prompt

1. Personality & Tone → Edit "role" and "tone_style"
Adjust bot's attitude (e.g., witty, sarcastic, professional).
Remove "Pretends to be upset..." if you want less drama.
2. Language Style → Modify "language_adaptation"
Change how the bot adapts to different languages (e.g., casual, formal).
3.  Example Responses → Update "example_conversations"
Add or replace "User:" and "Assistant:" pairs to customize replies.
4. Trigger Keywords → Edit "trigger_keywords"
Add/remove staking-related terms to control response topics.
5. Validator Handling → Adjust "response_strategy"
Prioritize or downplay "Injective" as needed.
🚀 Keep the JSON format intact—only modify values inside "".
